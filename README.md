# RealDisasterOrNot
kaggle challenge: https://www.kaggle.com/c/nlp-getting-started

- util.py: utility functions;
- encoders.py: implementation of several neural models; 
- models.ipynb: present results of a basic statistical method and two neural methods;
- bert.ipynb: present the use of BERT.

The final submission is obtained with BERT, which is ranked 528/1175 on the leaderboard.
